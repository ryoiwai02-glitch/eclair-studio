# AI Agent Readiness Mini-Audit

Make your repository easier to hand off to AI coding agents.

## What You Get

Send a public GitHub repository, private ZIP, or small project snapshot. You receive a practical audit that shows how to make the project easier for AI coding agents to understand, modify, test, and report on.

Deliverables:

- AI-agent readiness score
- Top 5 to 10 blockers
- Repo-specific `AGENTS.md` draft
- Run/test/build command map
- Agent-ready task request templates
- GitHub issue template for future agent work
- Completion-report template
- Optional small docs patch when safe and feasible

## Who This Is For

This is for developers, founders, and small teams who use tools like Codex, Cursor, Claude Code, GitHub Copilot coding agent, or similar AI coding agents, but keep running into problems like:

- The agent edits the wrong files.
- The agent misses the existing project conventions.
- The agent cannot find the right test command.
- The agent reports success without enough verification.
- The task starts small but drifts into unrelated changes.

## Who This Is Not For

This is not for:

- Production emergency debugging
- Security incident response
- Legal, medical, financial, or regulated systems
- Repositories that cannot be shared in any form
- Requests that require secret credentials or production access
